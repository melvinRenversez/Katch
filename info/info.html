Katch

Accueil Download

01 : Télécharge le JDK

-Rendez-vous à l'accueil, puis cliquez sur download.

-Sélectionnez le bouton download dans la page JDK indiquée en haut de page.

02 : Extraire le JDK

-Une fois le JDK téléchargé, il se trouve dans le dossier téléchargement.

-Le JDK est en ZIP. Il faut donc extraire le dossier.

-Une fois le JDK extrait, déplacer le JDK dans le dossier Document.

03 : Installer le JDK

-Une fois le JDK mis dans les documents, chercher dans votre barre de recherche "variable d'environnement pour votre compte".

- Cherchez la ligne nommée PATH, puis cliquez sur Modifier.

-Une fois dans cette page, cliquer sur le bouton parcourir.

-Cherchez parmi vos dossiers, le dossier document puis le dossier JDK.

-Puis cliquer sur ok pour toute les pages.

Maintenant, vous pouvez lancer l'application chat sans problème.
